# 指标口径字典

## 数据粒度

- `ecommerce_all`：五个月度表合并后的原始行为日志，一行代表一次行为记录。
- `ecommerce_clean`：全字段去重、时间转换和缺失值标准化后的行为日志。
- `ecommerce_daily_metrics`：日粒度汇总表，一天一行。

## 核心指标

| 指标 | 定义 | 关键口径 |
|---|---|---|
| 总行为数 | 清洗表记录数 | `COUNT(*)` |
| 活跃用户数 | 发生任意行为的去重用户数 | `COUNT(DISTINCT user_id)` |
| 活跃商品数 | 出现过的去重商品数 | `COUNT(DISTINCT product_id)` |
| 总会话数 | 非空去重会话数 | `COUNT(DISTINCT user_session)` |
| 购买用户数 | 至少发生一次 purchase 的去重用户数 | 不限制价格 |
| 购买会话数 | 至少包含一次 purchase 的非空去重会话数 | 不限制价格 |
| 购买事件数 | 清洗后 purchase 行为记录数 | 不是订单量 |
| 有效购买事件数 | purchase 且 `is_valid_price = 1` | 用于金额分母 |
| 交易金额估算 | 有效购买事件 price 之和 | 不是严格 GMV |
| 用户购买转化率 | 购买用户数 / 活跃用户数 | 用户口径 |
| 会话购买转化率 | 购买会话数 / 总会话数 | 会话口径 |
| 平均购买事件金额 | 交易金额估算 / 有效购买事件数 | 不是客单价 |

## 清洗标记

| 字段 | 含义 |
|---|---|
| `is_valid_price = 1` | price > 0，可参与金额计算 |
| `is_valid_price = 0` | price <= 0，保留行为但不参与金额计算 |

## 时间指标

- 日环比：`(当天 - 前一天) / 前一天 × 100%`。
- 7 日移动平均：当前日及前 6 行的简单平均。
- 7 日每日用户转化率均值：每日用户转化率的简单移动平均，不是跨日用户重新去重后的转化率。
